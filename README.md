# Vibe Habbit RPG

Gamified habit tracker that treats your life like an RPG.

Vibe Habbit RPG turns habits into quests, skills into stats and consistency into levels.

There is no final boss. The level cap keeps moving. Good luck.

## Features

- **Character Customization**  - Customize your pixel art character with heads, armor, weapons, and accessories.

- **Skills System**  - Create skills that gain XP and level up as you complete related habits. No hard cap, levels keep growing.

- **Habit Tracking**  - Add daily habits, link them to skills, and complete them like repeatable RPG quests.

- **XP & Leveling**  - Completing habits grants XP. Skills level up every 5 XP.

## How to run

- `docker compose up`
